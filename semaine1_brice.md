- synthese automatique reutilisable pour le rapport.

Pour executer le notebook :

1. deposer les fichiers Kaggle dans `data/azure/` ;
2. si besoin pour la suite du projet, deposer `ai4i2020.csv` dans `data/ai4i/` ;
3. ouvrir `Untitled.ipynb` ;
4. executer les cellules dans l'ordre.
